🎯 O Desafio
O teste consiste em validar duas aplicações distintas:
- Plataforma de e-commerce: [Sauce Demo](https://www.saucedemo.com/v1/)
- API de sistema de reservas de hotel: [Restful-Booker](https://restful-booker.herokuapp.com/apidoc/index.html)

# 📊 Níveis de Implementação
#### UI Testing (Sauce Demo)
##### Nível 1 (Obrigatório)
- Login com diferentes tipos de usuários = OK
- Ordenação e filtragem de produtos = OK
- Fluxo completo de compra = OK
- Remoção de itens do carrinho = OK
- Navegação entre páginas = OK
- Logout = OK

##### Nível 2 (Diferencial)
- Testes de responsividade
- Testes de acessibilidade
- Testes automatizados = OK

#### API Testing (Restful-Booker)
##### Nível 1 (Obrigatório)
- Autenticação básica
- CRUD de reservas
- Validação de campos obrigatórios

##### Nível 2 (Diferencial)
- Testes de performance
- Testes de segurança
- Automação via scripts
