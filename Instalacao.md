#### UI Tests
- Qq precisa pra rodar o cypress? node? alguma outra coisa?
- bah as instrucoes lá de como pegar no git e tals
- com o projeto pasteado bunitinho...
- Open a terminal via Terminal - New Terminal (there are many types of terminals, I used cmd)
- Navigate to UI tests with: cd "UI tests"
- Run this command to install Cypress
- npm install cypress --save-dev
- Now open Cypress with command: npx cypress open
- Select E2E Testing and wait for it to create additional files
- Resto depende de ter os arquivos e tals
